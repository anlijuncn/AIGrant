\section*{Milestones}

\begin{itemize}
  \item Q1--Q2: project initiation, approvals, and initial data setup.
  \item Q2--Q5: cohort assembly, quality control, and harmonization.
  \item Q4--Q8: core analysis or model development.
  \item Q7--Q9: validation and robustness checks.
  \item Q9--Q12: interpretation, dissemination, and final reporting.
\end{itemize}
